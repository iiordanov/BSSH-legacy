Virtual Keyboard

By default, BSSH is set to use Landscape mode. However, you may find that in Landscape mode, the virtual keyboard takes up a large part of your screen. Therefore, you may prefer to use BSSH with a virtual keyboard in Portrait mode in order to see more lines on the terminal.

To change this setting, at the Host List swipe from the top of the screen down and select Settings. Then, select your preference in Rotation mode.

On devices without a hardware keyboard, you may swipe from the bottom left corner diagonally inward to bring up the virtual keyboard. Please note that this applies to any program on the Blackberry platform - it is not BSSH specific.

Swipe from the bottom diagonally left or right in order to go back within the program.

How to Enter Control, Alt, Escape, Tab, and Function Keys

Tap the screen while connected to a host to see on-screen shortcuts to Ctrl, ESC, a table of symbols (SYM), and the virtual keyboard (a picture of a keyboard).

You can enter any key combination with BSSH and the virtual keyboard, but you must know how keys are mapped on a normal console. For instance, usually combinations of ALT+letter on a PC keyboard are actually mapped to sending, sequentially, ESC key then the letter.

Here are some important keys neither present on the virtual keyboard nor in an on-screen shortcut:

  • Tab = Ctrl + i
  • Up Arrow = Ctrl + p
  • Down Arrow = Ctrl + n
  • Left Arrow = Ctrl + b
  • Right Arrow = Ctrl + f
  • Home = Ctrl + a
  • End = Ctrl + e
  • Cut (after cursor) = Ctrl + k
  • Paste (at cursor) = Ctrl + y
  • Skip Forward Word = Esc + f
  • Skip Back Word = Esc + b
  • Function Keys = Ctrl + numbers 1-9

Note there are also screen gestures: see the ScreenGestures help topic for help on Page Up, Page Down, Scrolling, Switching hosts, and Going Back.
